// Описание диаграммы БД TXSTBXRD.
Схему создавал тут - https://dbdesign.online/

% Таблица "Project" || Содержит описание Тестового Проекта.
1. ID - первичный ключ.
2. Name - Имя проекта.
3. ScreenSaver - Шапка или фон проекта.


% Таблица "Consumers" || Содержит описание Работника у которого есть доступ к пространству проекта.
1. ID - первичный ключ.
2. Name - Имя.

% Таблица "Project_Consumers" || Реализует связть многие ко многим, между таблицами проекта и работника.
1-2 первичные ключи предыдущих страниц.
3. StartDate - Дата добавления работника к проекту.
4. EndDate - Дата удаления работника из проекта.
